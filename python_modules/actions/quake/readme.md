# Earthquake loads

From Wikipedia: seismic loading is one of the basic concepts of earthquake engineering which means application of an earthquake-generated agitation to a structure. It happens at contact surfaces of a structure either with the ground, or with adjacent structures, or with gravity waves from tsunami.
